Fuzzing with Code Fragments
========================================

简介
----------------------------------------
Fuzzing with Code Fragments 这篇论文介绍了一个名为 LangFuzz 的模糊测试器。 LangFuzz 利用给定的编程语言的语法来生成代码，由代码示例和变异引擎来创建测试用例。
